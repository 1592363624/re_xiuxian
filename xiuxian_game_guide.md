# 修仙游戏说明

这是一个基于 Telegram 群聊的长线文字修仙游戏。玩家从检测灵根开始，靠闭关、宗门、洞府、交易、斗法、副本和事件逐步提升境界。游戏具有较强的异步特征：很多收益来自每日命令、后台结算、限时队伍和随机奇遇。

## 1. 开始游戏

1. 发送 `.检测灵根` 进入修仙体系。
2. 用 `.我的灵根` 查看自己的灵根、境界、修为、灵石、道号等基础信息。
3. 用 `.储物袋` 查看背包物品。
4. 用 `.闭关修炼` 获取修为，这是前期最稳定的成长方式。

如果需要修改身份展示，可用 `.改道号 <新道号>`。如果存在多个角色或夺舍状态，可用 `.切换` 管理当前角色。

## 2. 核心成长循环

游戏的主循环是：

`闭关修炼 -> 获取修为和资源 -> 使用丹药/装备/宗门增益 -> 冲击境界 -> 解锁新系统`

前期重点：

- 每天做 `.闭关修炼`、`.宗门点卯`、`.宗门传功`。
- 尽早 `.拜入宗门 <宗门名>`，宗门提供贡献、宝库、任务和特色玩法。
- 用 `.储物袋` 检查丹药、材料、装备，再用 `.服用`、`.使用`、`.装备` 提升能力。
- 突破前确认状态，虚弱、心魔、道心问题会影响风险。

## 3. 境界与突破

境界从炼气开始，逐步进入筑基、结丹、元婴、化神、炼虚、合体、大乘、渡劫、真仙等阶段。

关键突破命令：

- `.冲击结丹`
- `.冲击元婴`
- `.冲击化神`
- `.飞升灵界`

元婴以后会出现更多阶段机制，如 `.元婴状态`、`.元婴出窍`、`.问道`、`.法相天地`、`.探寻裂缝`。失败可能触发虚弱、残魂、夺舍等流程，需要按系统提示使用 `.天机回溯`、`.夺舍重生`、`.重生 <编号>` 等命令处理。

## 4. 修炼方式

常规修炼使用 `.闭关修炼`，适合日常获取修为。高阶或长线挂机可使用 `.深度闭关`，之后通过 `.查看闭关` 查看进度，必要时 `.强行出关`。

悟道体系使用：

- `.静思悟道`
- `.中断悟道`

部分宗门、侍妾、星宫、天星宗、小世界、灵兽和法宝效果会影响修炼成功率、收益或失败惩罚。

## 5. 宗门玩法

宗门是中期以后最重要的长期系统之一。

基础流程：

1. `.宗门列表` 查看宗门。
2. `.拜入宗门 <宗门名>` 加入。
3. `.我的宗门` 查看身份。
4. 每日使用 `.宗门点卯`、`.宗门传功`。
5. 用 `.宗门悬赏` 接任务，用 `.提交任务 <任务名>` 提交。
6. 贡献可在 `.宗门宝库` 中用 `.兑换 <物品>` 兑换资源。

不同宗门有专属玩法。例如落云宗有灵眼之树，星宫有观星台和侍妾，天星宗有命盘，凌霄宫有天阶，阴罗宗和合欢宗有魔道/炉鼎体系。

## 6. 洞府与生产

洞府是个人经营系统。

基础步骤：

1. `.开辟洞府`
2. `.洞府`
3. 按需升级 `.升级灵脉`、`.升级静室`、`.升级丹房`、`.升级器室`、`.升级大阵`
4. 用 `.小药园`、`.播种 <种子>`、`.采药` 等命令经营药园

洞府还支持拜访、留言、访客、景观和商人：

- `.拜访洞府 [@用户]`
- `.洞府留言 <内容>`
- `.查看留言`
- `.查看访客`
- `.布置景观 <景观>`
- `.查看货品`
- `.购买商品 <编号>`

## 7. 装备、炼制与法宝

炼制系统使用 `.炼制 <物品> [数量]`。学习配方或功法通常使用 `.学习 <名称>`。

法宝系统包括装备、祭炼、本命、祭出、器灵等阶段：

- `.装备 <物品>`
- `.法宝`
- `.祭炼 <法宝>`
- `.本命 <法宝>`
- `.祭出 <法宝>`
- `.收宝 <法宝>`
- `.唤醒器灵 <法宝>`
- `.我的器灵`
- `.器灵试炼 <法宝>`

法宝和器灵会影响战斗、护主、试炼和部分副本表现。

## 8. 灵兽

灵兽是战斗和资源辅助系统。

常用流程：

1. `.寻觅灵兽`
2. `.我的灵兽`
3. `.灵兽状态 <灵兽>`
4. `.灵兽互动 <灵兽> 抚摸`
5. `.灵兽出战 <灵兽>`
6. `.喂养 <灵兽> <物品> <数量>`
7. `.灵兽放养 <灵兽>` 或 `.一键放养`
8. `.灵兽巡游 <灵兽>`
9. `.灵兽升星 <灵兽>`

灵兽拥有性情、心情、羁绊、忠诚、体力、通灵星级和兽魂。喂养、互动、放养、巡游会改变这些状态；心情和羁绊会影响偷菜、探渊等行动表现。放养中的灵兽可用 `.灵兽休息 <灵兽>` 手动召回，已到归来时间会直接结算收获，提前召回则不结算放养收获。灵兽满级后不会浪费经验，继续喂养、训练、巡游、偷菜或探渊会把经验按阶级凝成兽魂。兽魂可配合妖丹用 `.灵兽升星 <灵兽>` 进行通灵升星，提升战力，并在 3 星、5 星强化噬金虫、啼魂兽、六翼霜蚣等灵兽的招牌特性。灵兽还可以偷菜、探渊、孵化、寄养到小世界。不同宗门和玩法可能给灵兽额外收益。

## 9. 战斗与风险

PVP 主要有斗法、决斗、悬赏、封神台等。

- `.斗法 [@用户]` 是常规玩家战斗。
- `.决斗 <目标>` / `.对决 <目标>` 会进入回合对决，私聊中直接发送 `神通`、`防御`、`蓄力` 出招。
- `.战力 [@用户]` 和 `.战力对比 <目标A> <目标B>` 用于估算强弱。
- `.发布悬赏`、`.接单`、`.悬赏榜` 用于悬赏追杀。
- `.封神台`、`.设置防守`、`.挑战神君 <排名>` 是镜像排名战。

斗法失败可能带来虚弱、损失、因果或其他负面效果。境界差距过大时要谨慎出手。

当铺：

- `.当铺`
- `.当 <物品> <数量>`

股市：

- `.股市` / `.大盘`

## 11. 副本与组队

副本通常分为“开启、加入、进入、抉择、状态、奖励、解散”几个步骤。

常见副本：

- 虚天殿：`.开启虚天殿`、`.进入虚天殿`、`.虚天殿帮助`
- 昆吾山·封魔塔：`.开启昆吾山`、`.加入昆吾山`、`.进入昆吾山`、`.昆吾抉择`、`.昆吾山状态`、`.昆吾山帮助`
- 坠魔谷：`.开启坠魔谷`、`.坠魔谷状态`、`.坠魔谷帮助`
- 黄龙山：`.开启黄龙山`、`.黄龙山状态`、`.黄龙山帮助`
- 血色试炼：`.开启血色试炼`、`.血色试炼状态`、`.血色试炼帮助`
- 苍坤洞府：`.开启苍坤洞府`、`.苍坤洞府状态`、`.苍坤洞府帮助`
- 掩月抢亲：`.合成红绫婚帖`、`.开启掩月抢亲`、`.掩月抢亲状态`、`.掩月抢亲帮助`，这是苍坤洞府后的 3-5 人剧情副本；队长需元婴初期并成功通关过苍坤洞府，开本消耗【红绫婚帖】x1，队员需结丹后期
- 端午镇蛟：`.开启端午镇蛟`、`.端午投粽`、`.端午状态`、`.端午帮助`，必须满 10 人方可开打，成功入江后全员进入 23 小时冷却；需先在 LDC shop 购买【美味肉粽】并投入江中，若入江时无人投粽则立即失败并触发空舟惩罚，每人最多扣 388 灵石、500 修为；稀有掉落【五色丝】、【雄黄残晶】、【粽叶灵符】、【沧龙江镇蛟者】均可实际使用

副本期间同一话题可能会屏蔽其他修仙指令，优先按副本提示行动。队伍满员或进场窗口关闭后，迟到玩家通常无法加入。

昆吾山新版是固定 5 人剧情副本“昆吾山·封魔塔”，不再是旧版十层随机岔路。每位队员进本消耗 1 枚【昆吾通行令】，队长带队进入后按四幕推进：

1. 第一幕选择入山策略：用 `.昆吾抉择 路径1/路径2/路径3/路径4`，分别对应八灵尺光、化龙玺镇脉、黑风旗裂影、太阴火潮等路线。
2. 第二幕进入昆吾遗宫：系统随机给出 3 个遗迹选项，用 `.昆吾抉择 1/2/3` 决定取封纹、火种、牌影、残印或黑风旗残幅等收益。
3. 第三幕处理封魔塔三处阵眼：依次面对八灵尺、化龙玺、黑风旗阵眼，用 `.昆吾抉择 镇/夺/断` 在补封、夺宝和断禁之间取舍。
4. 第四幕自动进入玲珑封魔塔决战：队伍需要在有限回合内打掉塔心魔影，并把封印推进到目标值。

昆吾山会持续维护六个变量：封印、魔气、山禁、宝压、玲珑、士气。封印越高越容易镇塔，魔气过高或士气归零会失败；山禁影响推进压力，宝压代表夺宝反噬与爆发收益，玲珑代表塔影线索与神念清明。随时用 `.昆吾山状态` 查看当前阶段和变量。

## 15. 注意事项

- 大多数命令必须在指定群、指定话题或私聊场景中使用。
- 机器人会记录修仙指令并进行反挂机检测，删除命令、重复脚本化输入可能触发审判或验证。
- 虚弱、闭关、夺舍、元婴出窍、副本锁定等状态会限制可用命令。
- 高阶突破和副本有失败代价，先看帮助和状态再行动。
- 私聊导入档案前应确认来源，避免覆盖自己的角色数据。

## 16. 世界事件与大型公共战斗

除了副本，项目里还有公共事件，通常由后台或管理员开启，所有符合条件的修士都可参与。

青元子世界 Boss 使用：

- `.世界boss` 查看状态。
- `.世界boss榜` 查看贡献榜。
- `.世界boss奖励` 查看奖励池。
- `.讨伐青元子 强攻` 主要打伤害。
- `.讨伐青元子 破幡` 推进破幡，削弱 Boss 机制。
- `.讨伐青元子 镇魂` 降低魔压。
- `.讨伐青元子 护阵` 恢复阵势。

这个玩法不是单纯堆输出。重复同一行动会有惩罚，阶段中有幡魂、魔压、阵势、点名、锁血等机制。队伍里需要有人破幡、镇魂、护阵，否则强攻收益会下降，甚至导致整体压力失控。

妖兽入侵分两段：

1. 捐献阶段：使用 `.贡献 <物品名> <数量>` 完成锁灵大阵，`.贡献查询` 查看进度。
2. 战斗阶段：达标后使用 `.斩妖` 攻击，`.妖兽入侵` 查看战况。

`.妖兽入侵帮助` 和 `.妖兽入侵奖励` 会给出当期说明。战斗有冷却和聚合战报，系统不会每次攻击都刷屏。

慕兰烽烟发生在天南边境战线。慕兰法士借圣灯压阵南下，黄龙山旧部与各派修士在前线固守法阵、截断粮道，并搜寻苍坤旧禁残线。它每天会刷新一张“军议战局”，玩家应先看战局再决定支援路线：

- `.慕兰烽烟` / `.慕兰军议` 查看个人战线状态、今日是否已支援、累计边境军功、今日密令、险棋窗口和粮道路线。
- `.支援慕兰 斥候` 稳健探路，偏基础材料。
- `.支援慕兰 破灯` 克制慕兰圣灯，低概率获得【慕兰圣灯残焰】。
- `.支援慕兰 护阵` 风险最低，偏灵石和阵旗残片。
- `.支援慕兰 奇袭` 收益和风险更高，偏妖丹、蛇胆和稀有战利品。

今日军议会给出三个关键提示：密令路线押中后有额外军功和收益；险棋路线风险更高但成功额外立功；粮道路线灵石收益更高。累计边境军功达到 7、21、49、81 时会自动发放里程碑奖励，最高可得专属徽章【天南边军】。

慕兰谍影把战线做成“真假情报”小游戏：

- `.慕兰谍影` 查看今日军报匣。
- `.搜集军报` 每日取得三条真假混杂的前线消息。
- `.辨报 <编号>` 研判某条军报的可信度和线索。
- `.公开军报 <编号>` 将你认定的军报交给前线。真军报会给边境军功，并强化今日对应支援路线；假军报会让今日支援承受额外警戒。

军功司让边境军功有长期用途：

- `.军功司` / `.我的军衔` 查看军衔、累计军功、可用军功和兑换物。
- `.兑换军功 <物品>` 消耗可用军功兑换回灵丹、黄龙军功牌、慕兰圣灯残焰、神识玉简等。
- 军衔看累计军功，不会因为兑换而倒退；兑换只消耗可用军功。

慕兰战线也接入了灵兽、残图和法宝三条支线，形成一个小循环：

- `.灵兽边境` 查看当前是否有灵兽巡边；`.灵兽巡边 <灵兽名> <斥候/护粮/袭营>` 派出灵兽，稍后用 `.巡边归来` 结算。斥候偏残图路线，护粮偏灵石补给，袭营风险更高但军功和稀有材料更好。
- `.残图匣` 查看四类苍坤残片；`.拼残图` 用四类残片拼成完整残图；`.按图探禁` 消耗完整残图，进入苍坤旧禁边缘，获得旧禁材料、修为和军功。
- `.刻印状态` 查看今日临战刻印；`.临战刻印 <法宝名> <破灯/护阵/潜行>` 消耗材料给已有法宝施加一次性刻印。破灯适配 `.支援慕兰 破灯`，护阵适配 `.支援慕兰 护阵`，潜行适配 `.支援慕兰 斥候` 或 `.支援慕兰 奇袭`。

推荐顺序是先看 `.慕兰军议`，再决定是否用灵兽巡边补残图或用材料刻印法宝，最后执行当天的 `.支援慕兰`。刻印只在当天匹配路线时自动触发一次。

坐化遗府用于处理退坑修士的可转移资产，由管理员开启：

- `.遗府` 查看当前遗府。
- `.转动遗府` 每期只能分宝一次。
- `.遗府榜` 查看分宝记录。
- 管理员先用 `.admin 遗府预览 @用户` 查看可转移资产和合格玩家估算。
- 管理员确认后用 `.admin 开启遗府 @用户 [小时]` 开启活动。
- 管理员可用 `.admin 遗府状态` 查看后台状态，用 `.admin 关闭遗府` 手动结束。

遗府只分配储物袋里的普通材料和普通物品。剧情物、徽章、绑定物、带器灵或特殊字段的实例、已装备/祭炼/本命法宝会跳过。参与资格会按近期修仙活跃、首次记录时间、总指令数、总修为、频道身份绑定和同主魂唯一领取来限制，防止临时小号批量领取。

## 22. 隐藏/事件式命令

有些命令不是随时可用，而是事件消息出现后才有意义：

- `.献上魂魄` / `.收敛气息`：通常回复特殊试炼或事件消息。

这些命令通常要“回复系统消息”或处于特定状态才会生效。看到事件提示时优先按提示输入，平时单独发送可能没有效果。

## 23. 大衍诀与傀儡路线

大衍诀是最新命令文档中单独成章的后期成长线，核心价值是提高神识操控、傀儡运用、裂缝探索、节点稳定和飞升前置能力。它不是单纯战斗技能，而是“神识 + 傀儡 + 高阶副本 + 飞升”的联动系统。修炼后会按层数提供有效神识倍率，一层到五层约为 x1.30、x1.60、x2.00、x2.45、x3.00，影响探查、斗法神识威压、御宝预算、傀儡牵丝和神识类成功率判定。

入门顺序：

1. 获取大衍诀残篇后，用 `.参悟大衍诀`、`.大衍诀` 或 `.修炼大衍诀` 查看和推进层数。
2. 获得傀儡图谱后，用 `.参悟图谱 <图谱名>` 解锁制造。
3. 用 `.傀儡` 查看傀儡工坊。
4. 用 `.制造傀儡 <类型>` 制造机关木傀、铁甲战傀、五行灵傀、影傀或大衍灵傀。
5. 用 `.傀儡出战 <编号>` 设置战斗用途，或 `.傀儡护法 <编号>` 设置闭关护法。
6. 用 `.傀儡淬炼 <编号>` 提升等级，用 `.傀儡维修 <编号>` 恢复耐久。
7. 不需要的傀儡先用 `.傀儡回收 <编号>` 预览，再用 `.傀儡回收 <编号> 确认` 回收。

大衍诀残篇来源要分清：凝识和分念来自 LDC shop；控傀来自昆吾山；千机来自苍坤洞府；衍神来自青元子世界 Boss。飞升灵界需要大衍诀五层·衍神，所以后期不要忽略世界 Boss 和高阶副本。

傀儡图谱也有来源差异。影傀图谱来自虚天殿和玄骨高阶分支；大衍灵傀图谱来自青元子世界 Boss。万年养魂木可通过 `.掌天瓶 养木` 把 30 份养魂木配合掌天绿液逆养获得，也会在苍坤和高阶 Boss 中低概率出现；新版昆吾山主要产出封魔塔相关封纹、火种、牌影、残印和法宝遗响。

灵树每小时会出现不同环境，例如干渴需水、根基松动需土、生机萎靡需木、寒气侵蚀需火/雷、枝叶杂乱需金/风。灌溉时系统会根据你的灵根亲和自动注入最合适的五行灵气。

云梦灵眼定脉是新增玩法，不替代整套成熟、守山、采摘和奖励体系，只替代日常“灌溉推进”的操作方式。每名弟子每日有 6 次定脉令，其中冲脉最多 2 次。今日脉象分为主脉、辅脉、逆脉和平脉：选中主脉/辅脉更稳，强冲逆脉收益低且更容易产生浊息和紊乱。

定脉四种动作：

- 注灵：常规推进成熟度和主榜功绩，适合选择自己灵根能牵引且命中今日主脉/辅脉的五行。
- 固脉：成熟度较低，但提升脉稳和护山底蕴，适合多灵根补位、伪灵根和废灵根稳局。
- 净浊：主榜功绩较少，主要降低浊息/紊乱并计入净化榜，适合伪灵根、废灵根和想走稳根奖励的弟子。
- 冲脉：高风险高收益，消耗更多修为，脉稳低于 25 不可用；单灵根命中主脉时爆发最高，多灵根会因分脉而折减，伪灵根较难冲脉，废灵根不可冲脉。

灵根定位已经重新平衡：

- 单灵根和天灵根匹配时收益最高，适合看准时辰冲榜；不匹配时收益大幅降低，并会明显增加浊气和紊乱。
- 多属性灵根覆盖面更广，适合稳定补位；匹配收益低于单灵根，不匹配代价中等。
- 伪灵根收益平稳，擅长缓解浊气。
- 废灵根成长收益较低，但净化能力最强，可以稳定降低浊气和紊乱。

浊气和紊乱是本轮灵树的公共状态。旧灌溉下错误灌溉会增加浊气/紊乱；新定脉下错选逆脉、无亲和强行注灵或冲脉失败也会增加浊息和紊乱。匹配灌溉、固脉、净浊、木髓温养和献枝护宗都能不同程度净化。浊气过高会降低灵眼之液和【一截灵眼之树】概率，削减采摘修为；达到极高时会让果品降阶。紊乱过高会让阶段灵纹有概率凝成杂纹，影响最终果实品质。

榜单分为两类：

- 分枝榜决定【一截灵眼之树】相关奖励。化神及以上可以正常参与灵树和采摘，但不计入分枝榜。
- 净化榜按本轮净化贡献排名，奖励木髓碎片和灵眼之液，适合伪灵根、废灵根和愿意奉养灵树的弟子。

采摘结算时，主榜贡献仍会影响修为、木髓和灵液等收益；分枝榜影响【一截灵眼之树】；净化榜给额外稳根奖励。若个人错灌积累的污染债较高，会额外降低自己的树枝概率。若全宗浊气极重，即使位列前三也可能无法稳得树枝，但会获得灵眼之液和木髓补偿。

## 32. 阶段路线建议

前期目标是稳定进入系统：检测灵根、加入宗门、闭关、做宗门任务、开洞府和药园。不要急着 PVP，先把 `.我的灵根`、`.储物袋`、`.宗门列表`、`.宗门悬赏`、`.小药园` 熟悉。

中期目标是建立资源循环：宗门贡献换关键材料，洞府药园产出炼制材料，灵兽出战或放养，法宝开始祭炼，万宝楼和当铺补齐缺口。此时可以用 `.战力`、`.切磋木人`、`.我的阵法` 检查战斗准备。

后期目标是系统联动：第二元神、神识、小世界、掌天瓶、虚天鼎、大五行幻世轮、大衍诀、傀儡、世界 Boss 和高阶副本会互相卡资源。飞升线尤其依赖 `.搜寻节点`、`.定星`、`.飞升灵界`、大衍诀五层·衍神、法则材料和关键丹方。
