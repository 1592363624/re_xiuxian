# 详细数据设计文档（含数值平衡与界面布局）

# 一、核心世界观与题材定位

本游戏严格遵循《凡人修仙传》原著世界观框架，以“凡人逆袭、步步为营”为核心题材，构建涵盖人界、灵界、仙界的跨域修仙体系。玩家将化身出身平凡的修士，从炼气期起步，历经筑基、结丹、元婴等境界突破，在天南、乱星海、雷鸣大陆等经典地域探索秘境、争夺资源、结交道友或遭遇仇敌，最终渡劫飞升、冲击道祖之境。游戏核心玩法聚焦“数值成长策略性”“资源获取平衡性”“玩家交互协作性”,强化策略抉择与数值规划的乐趣。

# 二、核心数值体系设计（精确到具体数值）

## 2.1 境界体系与突破数值

### 2.1.0 灵根初始化分配系统

玩家创建角色时，系统自动随机分配1种基础灵根（金、木、水、火、土），每种灵根分配概率均等（各20%），无重复分配情况。灵根为角色核心先天属性，决定初始属性倾向、功法适配性及战斗属性相克效果，后续可通过“洗灵丹”“灵根进化液”等特殊道具重置或提升灵根品质（基础灵根→变异灵根→天灵根），但基础灵根属性（金木水火土）不可变更。

#### 2.1.0.1 灵根初始属性加成

不同灵根对应不同初始属性加成，确保先天差异合理且不打破平衡（加成数值控制在同境界基础属性的5%~8%），具体如下：

|灵根类型|初始属性加成|适配功法倾向|
|---|---|---|
|金灵根|攻击+4、防御+2|金属性攻击功法、防御型功法|
|木灵根|气血+30、灵力+8|木属性治疗功法、持续伤害功法|
|水灵根|速度+3、灵力+10|水属性控制功法、高速输出功法|
|火灵根|攻击+5、速度+2|火属性爆发功法、范围攻击功法|
|土灵根|防御+4、气血+25|土属性防御功法、控场功法|
#### 2.1.0.2 五行相克数值规则

遵循经典五行相克逻辑（金克木、木克土、土克水、水克火、火克金），相克关系仅作用于战斗伤害计算，通过固定系数影响最终伤害，与现有属性相克体系融合，具体规则如下：

- 克制方攻击被克制方时，伤害×1.3（与现有属性相克系数一致）；示例：金灵根玩家使用金属性功法攻击木灵根玩家，最终伤害在原有计算基础上额外提升30%。

- 被克制方攻击克制方时，伤害×0.7；示例：木灵根玩家使用木属性功法攻击金灵根玩家，最终伤害在原有计算基础上降低30%。

- 无相克关系（如金灵根vs水灵根、木灵根vs火灵根等）时，伤害系数×1.0（无增减）。

- 灵根相克优先级高于其他属性相克（如雷克所有），若同时触发两种相克关系，取最高系数（不叠加）；示例：金灵根玩家攻击木灵根玩家，同时玩家装备雷克木属性法宝，此时仅触发灵根相克的1.3倍伤害，不叠加雷克木的1.3倍系数。

## 2.1.1 年龄增长系统规则

玩家年龄增长与游戏内核心玩法深度绑定，遵循“现实时间映射+玩法时长折算”原则，确保增长逻辑合理且贴合修仙世界观，具体规则如下：

### 2.1.1.1 年龄增长触发条件与换算标准

- 修炼增长：玩家主动修炼或开启自动修炼时，按“1现实分钟=1游戏天”折算年龄增长，即每修炼1现实分钟，角色年龄增加1天；使用修炼加速道具不改变年龄增长速率（仅提升修为获取速度）。示例：修炼1现实小时（60分钟），角色年龄增长60天（约2个月）。

- 任务/秘境增长：完成主线、支线、日常任务或探索秘境时，根据任务/秘境的预设耗时折算年龄，具体为“任务/秘境预设耗时（现实分钟）= 年龄增长天数”。示例：完成预设耗时30分钟的“越国山脉秘境探索”任务，角色年龄增长30天。

- 离线增长：玩家离线期间，系统按“1现实小时=1游戏天”自动折算年龄增长，每日离线年龄增长上限为24天（对应24现实小时），避免离线过久导致年龄过快增长。

### 2.1.1.2 年龄增长约束与特殊机制

- 年龄冻结：使用“驻颜丹”“岁月静止符”等特殊道具时，可冻结年龄增长，冻结时长按道具说明计算（如1颗驻颜丹冻结7天年龄增长）。

- 延寿机制：当角色年龄达到当前境界寿命上限时，可使用对应品质的延寿丹（炼气期用低阶延寿丹、筑基期用中阶延寿丹等）提升临时寿命，1颗低阶延寿丹可增加10年寿命，中阶增加30年，高阶增加100年，每种延寿丹每日最多使用1颗。

- 境界突破重置：突破至更高境界时，年龄不重置，但会解锁更高的寿命上限（如从炼气期突破至筑基期，寿命上限从120岁提升至200岁），同时解除当前年龄达到原上限的负面状态。

注：1. 角色年龄仅影响寿命相关状态，不直接关联属性数值，确保同境界玩家不因年龄差异产生数值差距，符合“同境界数值差距≤20%”的核心平衡原则；2. 初始年龄为0岁，突破小阶段后自动解锁对应阶段的寿命上限，无需额外激活；3. 若在某小阶段内年龄达到上限，优先推荐突破小阶段（突破后寿命上限自动提升），若暂时无法突破，可使用低阶延寿道具应急。

## 2.2 核心属性与战斗数值平衡

核心属性分为基础属性、衍生属性，所有属性数值严格遵循“境界主导+装备/功法辅助”原则，确保同境界玩家数值差距控制在20%以内，避免碾压式差距。

### 2.2.2 战斗伤害计算公式（核心平衡公式）

最终伤害=（攻击者攻击×1.2 - 防御者防御×0.8）×属性克制系数×功法加成系数×法宝加成系数 - 防御者特殊减免
说明：
1. 若（攻击者攻击×1.2 - 防御者防御×0.8）≤0，则最终伤害=攻击者攻击×0.3（保底伤害，避免“零伤害”僵局）；凡人阶段保底伤害为攻击者攻击×0.2
2. 功法加成系数：白阶功法×1.0、绿阶×1.1、蓝阶×1.25、紫阶×1.4、橙阶×1.6（功法品质数值严格控制，同境界最高仅能使用紫阶功法；凡人阶段仅可学习白阶基础功法，加成系数×0.8）
3. 法宝加成系数：1阶法宝×1.05、2阶×1.15、3阶×1.3（筑基期最高3阶，金丹期最高5阶，每阶加成差距≤0.15；凡人阶段无法装备法宝，无此加成）
4. 速度影响补充：战斗中出手顺序判定优先于伤害计算，速度更高者先出手；若双方速度相同，则随机判定出手顺序

### 2.2.3 回合制战斗功能详细设计

回合制战斗以“策略抉择+数值博弈”为核心，适配修仙世界观，分为单人PVE（秘境、试炼、怪物讨伐）、双人/多人组队PVE、玩家对战（PVP）三种模式，严格遵循“回合交替、行动判定、效果结算”闭环流程，确保战斗公平性与策略性。

#### 2.2.3.1 核心战斗规则

1.  回合定义：单回合为“双方依次完成1次行动”，回合结束后进入下一轮，直至战斗结束（某一方气血归0、主动认输或触发特殊战斗终结条件）；单场战斗无回合上限，但高阶秘境战斗（金丹期及以上）若超过50回合，双方将进入“灵力枯竭”状态，每回合额外损失10%灵力，灵力归0后每回合损失5%气血，强制加速战斗进程。
2.  出手顺序判定：优先按速度属性排序，速度差值≥20时，速度更高者100%先手；速度差值＜20时，按“速度值/100”计算先手概率（例：A速度120，B速度110，A先手概率=120/230≈52.2%，B先手概率≈47.8%）；若速度完全相同，随机判定先手（各50%概率）；组队战斗时，同一队伍内成员出手顺序可提前预设（默认按速度排序），也可手动调整（每回合战斗前可修改1次）。
3.  行动选择：每回合玩家可选择1项核心行动+任意数量辅助行动（辅助行动无次数限制，但部分道具/技能有冷却回合）；核心行动包括“普通攻击、法术攻击（消耗灵力）、法宝释放（消耗灵力+冷却）、防御（本回合减免20%伤害）、逃跑（仅PVE可用，成功率随境界差变化）、认输（PVP/PVE通用，直接判定失败）”；辅助行动包括“使用丹药（补血/补灵力/加buff）、切换功法、更换法宝、激活灵根天赋（每场战斗限1次，持续3回合）”。
4.  冷却机制：法术、法宝、灵根天赋等行动存在冷却回合，冷却期间无法再次使用，冷却倒计时随回合推进自动减少；1阶法术冷却1回合，2阶法术冷却2回合，3阶及以上法术冷却3~5回合；1~3阶法宝冷却2~3回合，4~5阶法宝冷却4~6回合；灵根天赋冷却10回合（每场战斗仅激活1次，冷却结束后仍不可再次激活）。
5.  战斗终结条件：① 单方所有角色气血归0（PVE为怪物气血归0，玩家获胜；玩家气血归0，战斗失败）；② 玩家主动认输/逃跑成功（逃跑成功无惩罚，认输失败需承受对方1次无减免攻击）；③ 触发特殊剧情（如秘境倒计时结束、击败核心BOSS等）；④ 高阶战斗中某一方灵力枯竭且气血≤10%，可触发“认输保护”，自动判定失败但无额外惩罚（避免极端消耗战）。

#### 2.2.3.2 回合制战斗流程（单回合详细拆解）

1. **回合初始化（0~1秒）**：系统结算上一回合残留效果（如持续伤害、buff/debuff倒计时），更新双方气血、灵力数值，刷新可行动列表（解除冷却完毕的技能/法宝）；若某一方触发“濒死保护”（气血≤10%），系统自动提示“可使用急救丹药，本次行动优先级提升”。

2. **出手顺序判定（1~2秒）**：根据双方速度属性计算先手概率，生成本次回合出手顺序，在界面顶部展示“本次回合出手顺序：玩家A→怪物B→玩家C”，组队战斗时按队伍预设顺序展示。

3. **依次执行行动（2~10秒/角色）**：按判定顺序依次执行各角色行动，每步行动后即时结算效果：
- 攻击类行动（普通/法术/法宝）：计算最终伤害（按2.2.2公式），扣除防御者气血，触发属性相克特效（如金克木时，防御者气血扣除时额外显示“金克木！伤害提升30%”）；
- 防御类行动：玩家选择防御后，本回合受到的所有伤害减免20%，同时灵力恢复10点（凡人阶段无灵力恢复）；
- 辅助类行动：使用丹药即时恢复气血/灵力（1颗低阶补血丹恢复100点气血，1颗低阶补灵丹恢复50点灵力），使用后扣除对应道具；切换功法/法宝即时生效，新功法/法宝加成替代原有加成；
- 逃跑行动：PVE中逃跑成功率=（玩家速度-怪物速度）/100 + 基础成功率30%，成功率上限80%，下限10%；逃跑失败时，玩家承受怪物1次普通攻击（无减免），且下一回合无法再次选择逃跑。

4. **回合结算（10~11秒）**：系统汇总本回合双方气血、灵力变化，展示“本回合总结：玩家A造成伤害280点，承受伤害120点；怪物B造成伤害120点，承受伤害280点”，同时更新buff/debuff状态（如持续伤害效果，本回合扣除目标50点气血，剩余2回合持续时间）。

5. **回合切换（11~12秒）**：判断是否满足战斗终结条件，若满足则进入战斗结束结算；若不满足则进入下一回合，重复上述流程，界面顶部回合数+1（如“当前回合：3/50”）。

#### 2.2.3.3 组队回合制战斗特殊规则

1.  队伍配置：组队战斗支持2~4人组队，每人可携带1件主战法宝+2件辅助法宝，主战法宝可主动释放，辅助法宝自动触发（如“防御型辅助法宝，每回合为队友增加10点防御，持续1回合”）；队伍需设置1名队长，队长可调整队伍出手顺序、开启“队伍阵法”（筑基期及以上可解锁，阵法持续5回合，冷却10回合，为队伍提供全属性+5%~10%加成）。
2.  协同机制：组队战斗中可触发“协同攻击”（同一回合内，2名及以上玩家攻击同一目标，最后攻击的玩家造成1.1倍伤害）、“协同防御”（1名玩家选择防御时，可为相邻队友分担30%伤害）；触发协同行动后，队伍成员获得“默契值”，默契值累积至100点可解锁“终极协同技能”（全队伍同时发动1次攻击，伤害为各自普通攻击的1.5倍，无灵力消耗，冷却15回合）。
3.  战败惩罚：组队战斗失败时，所有队员损失当前修为10%，年龄增加5岁；若队长战败，额外损失1件常用道具（随机）；若战斗中某一队员主动逃跑，剩余队员承受的伤害提升20%，逃跑队员无惩罚但扣除10点默契值（默契值最低为0）。

#### 2.2.3.4 回合制战斗数值平衡（新增补充）

1.  持续伤害/恢复数值：持续伤害效果（如木属性法术、毒系法宝）每回合伤害=攻击者攻击×0.3~0.5，持续2~3回合，总伤害不超过攻击者攻击×1.5；持续恢复效果（如木属性治疗功法、补血类法宝）每回合恢复=施法者攻击×0.2~0.4（木灵根玩家额外提升20%恢复量），持续2~3回合，总恢复不超过施法者攻击×1.2。
2.  buff/debuff数值控制：攻击类buff（如“烈火加持”）提升攻击10%~15%，持续2~3回合；防御类buff提升防御10%~15%，持续2~3回合；debuff（如“减速”）降低目标速度20%~30%，持续1~2回合；所有buff/debuff效果不可叠加，同一类型效果后释放者覆盖前释放者。
3.  跨境界战斗平衡：低境界玩家挑战高境界目标时，每相差1个大境界（如炼气期vs筑基期），低境界玩家伤害降低30%，承受伤害提升30%；高境界玩家挑战低境界目标时，伤害降低50%，承受伤害提升50%（避免高境界碾压低境界）；同境界不同小阶段战斗，伤害/防御差异控制在10%以内。
4.  灵根天赋数值：金灵根天赋“金刃护体”（激活后3回合内，每回合减免10%伤害，触发1次反击，反击伤害=攻击×0.3）；木灵根天赋“生机盎然”（激活后3回合内，每回合恢复气血=攻击×0.4，免疫毒系debuff）；水灵根天赋“水幕屏障”（激活后3回合内，降低目标速度25%，自身灵力恢复速度提升20%）；火灵根天赋“烈焰爆发”（激活后3回合内，攻击提升15%，法术伤害额外+10点）；土灵根天赋“大地守护”（激活后3回合内，防御提升15%，承受攻击时有30%概率触发“眩晕”目标1回合）；雷灵根天赋“雷霆一击”（激活后1回合内，攻击提升30%，触发“雷克所有”效果，伤害×1.3，冷却15回合）。
